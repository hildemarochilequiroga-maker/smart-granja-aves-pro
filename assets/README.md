# 📁 Estructura de Assets - Granja Total

## Organización de Carpetas

```
assets/
├── images/
│   ├── logo/           # Logos de la aplicación
│   ├── onboarding/     # Imágenes para pantallas de bienvenida
│   ├── illustrations/  # Ilustraciones y gráficos decorativos
│   ├── backgrounds/    # Fondos y patrones
│   └── icons/          # Iconos personalizados (PNG/SVG)
├── fonts/              # Fuentes personalizadas
└── animations/         # Animaciones Lottie (JSON)
```

## Formatos Recomendados

- **PNG**: Para logos e iconos con transparencia
- **JPG**: Para fotografías y fondos
- **SVG**: Para gráficos vectoriales escalables
- **JSON**: Para animaciones Lottie

## Resoluciones

### Para pantallas Android/iOS
- **1x**: Base (mdpi)
- **2x**: Resolución media-alta (xhdpi)
- **3x**: Resolución alta (xxhdpi)

### Tamaños recomendados
- **Logo principal**: 512x512 px (PNG transparente)
- **Splash screen**: 1080x1920 px
- **Ilustraciones**: 800x600 px
- **Iconos**: 128x128 px
