## Especificaciones de Imagen

- **Formato**: PNG o WebP (preferido)
- **Tamaño**: 400x300 píxeles (aspect ratio 4:3)
- **Fondo**: Preferiblemente transparente o blanco
- **Calidad**: Alta resolución para pantallas móviles
