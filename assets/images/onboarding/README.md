# 👋 Onboarding - Imágenes de Bienvenida

## Archivos Requeridos

### Pantallas de Onboarding (3-4 pantallas)
```
onboarding_1.png          # Primera pantalla - Introducción
onboarding_2.png          # Segunda pantalla - Características principales
onboarding_3.png          # Tercera pantalla - Beneficios
onboarding_4.png          # Cuarta pantalla (opcional) - Call to action
```

### Welcome Screen
```
welcome_background.png    # Fondo para pantalla de bienvenida
welcome_hero.png         # Imagen hero principal
```

### Especificaciones
- **Tamaño**: 800x600 px o 1080x810 px
- **Formato**: PNG o JPG
- **Estilo**: Ilustraciones coloridas, modernas y amigables
- **Orientación**: Vertical u horizontal según diseño

## Temas Sugeridos

1. **Onboarding 1**: Granja con pollos (introducción)
2. **Onboarding 2**: Control de inventario (funcionalidad)
3. **Onboarding 3**: Gráficos/reportes (análisis)
4. **Onboarding 4**: Persona usando celular (facilidad de uso)

## Ejemplo de Uso

```dart
Image.asset(
  'assets/images/onboarding/onboarding_1.png',
  fit: BoxFit.contain,
)
```
