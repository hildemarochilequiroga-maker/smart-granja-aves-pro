### Especificaciones
- **Tamaño**: 400x300 px a 600x450 px
- **Formato**: PNG con transparencia
- **Estilo**: Flat design, líneas suaves, colores corporativos
- **Uso**: Mejorar UX en pantallas vacías o de error

## Paleta de Colores

Usar colores consistentes con el tema de la app:
- Primario: Verde/Azul (agricultura)
- Secundario: Naranja/Amarillo (energía)
- Acento: Colores vibrantes para CTAs

## Ejemplo de Uso

```dart
Image.asset(
  'assets/images/illustrations/empty_state_inventory.png',
  width: 200,
  height: 150,
)
```
